# EduBridge – Scholarship Discovery Platform

## Overview

EduBridge is a responsive web application designed to help students discover government and private scholarships in India. The platform organizes scholarship information in one place, allowing users to browse opportunities, filter scholarships based on eligibility, and access detailed application information through an intuitive interface.

## Features

* Responsive and modern user interface
* Light and Dark theme support
* Scholarship eligibility filtering
* Government and private scholarship listings
* Detailed scholarship information pages
* Login and Sign Up interface
* Mobile-friendly navigation
* Clean and user-friendly design

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla)
* Google Fonts (Outfit)

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any modern web browser.
4. Navigate through the website using the navigation bar.

No additional installation or server setup is required.
